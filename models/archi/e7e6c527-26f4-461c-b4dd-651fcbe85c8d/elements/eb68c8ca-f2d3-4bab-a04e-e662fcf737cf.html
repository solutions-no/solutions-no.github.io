Data Set ()
DESCRIPTION: A Data Set is a collection of data, published or curated by a single agent, and available for access or download in one or more formats. Source: W3C http://www.w3.org/TR/vocab-dcat/#class-dataset INTEROPERABILITY SALIENCY: The Data Set ABB is salient for semantic interoperability because it provides a collection of data to be shared or exchanged between ICT systmes as stated in the EIF recommendation n° 30: "Perceive data and information as a public asset that should be appropriately generated, collected, managed, shared, protected and preserved." EXAMPLES: The following implementation is an example on how this specific Architecture Building Block (ABB) can be instantiated as a Solution Building Block (SBB): European Union Data Portal - The European Union Open Data Portal (EU ODP) gives you access to open data published by EU institutions and bodies. All the data you can find via this catalogue are free to use and reuse for commercial or non-commercial purposes. https://data.europa.eu/euodp/data/dataset
